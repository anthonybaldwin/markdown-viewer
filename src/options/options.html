    Why it’s safe

    Nothing in a document can run

    Markdown is parsed, then every byte is passed through DOMPurify before it reaches the page: no scripts, no inline event handlers, no remote-loading frames, no javascript: links. Math is rendered by KaTeX after sanitizing (it executes no code), and diagrams render in Mermaid’s strict mode and are scrubbed again before display. Everything is bundled locally — no network calls, no telemetry.
